# AirCrack_V2

🚀 **AirCrack_V2** adalah tools eksperimen untuk analisis jaringan WiFi dan pembelajaran keamanan jaringan.
Project ini dibuat untuk tujuan **edukasi dan research** di bidang cybersecurity.

---

## ⚡ Features

* 📡 Network scanning
* 📶 Packet monitoring
* 🔍 WiFi network analysis
* ⚙️ Modular system (cli / core / img)

---

## 📂 Project Structure

```
AirCrack_V2
│
├── cli/      # Command line interface
├── core/     # Core logic tools
├── img/      # Assets / images
└── README.md
```

## 🛠 Installation

Clone repository:

```bash
git clone https://github.com/kenzzSploit/AirCrack_V2.git
cd AirCrack_V2
```

Run tools:

```bash

```

atau

